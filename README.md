Dataset generation codes.zip contains all the matlab codes for generating training and validation datasets. When 
"Dataset_generation_main_code.m" is run, it generates training and validation datasets for each of the 210 unique sections 
from the four .csv files. 
The "Trained network weights.zip" contains the weights of the 210 trained networks. 
The experimental validation strain fields are in "DIC data.zip" which contains strain fields obtained by stretching in both
x and y  directions
